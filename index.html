How it works
A simple, repeatable workflow you can trust.

Workflow

  1. Select a source folder containing the files you want to analyse.
  2. Run an audit to identify duplicate groups without moving anything.
  3. Review the audit log to confirm the results look sensible.
  4. Run the dedupe pass to copy duplicates into a dedicated folder.
  5. Inspect the copied files and delete/archive them when you’re ready.

What it doesn’t do

  • Does not auto‑delete files behind your back.
  • Does not modify file contents.
  • Does not require admin rights or install services.
  • Does not touch anything you didn’t explicitly point it at.
FAQ
A few quick answers.
Is it safe to run on my main photo library?
Yes. EverythingDeduper only copies duplicates into a separate folder. You can verify everything before deleting anything.
Does it require installation?
No. It’s a portable EXE. Download, extract (if using the ZIP), and run.
Which versions of Windows are supported?
EverythingDeduper is designed for Windows 10 and Windows 11.
Where can I report any issus?
You can report issues by using tracker live on GitHub: github.com/carlwaring/EverythingDeduper.
