\# Changelog

All notable changes to Everything Deduper will be documented in this file.



\## \[1.2.0] – 2026‑04‑27

\### Added

\- \*\*Stop button\*\* for Auto‑Dedupe mode, with proper enable/disable logic.

\- \*\*Interruptible auto‑dedupe workflow\*\*, allowing the user to safely halt long operations.

\- \*\*New global flag\*\* to track auto‑dedupe running state.

\- \*\*UI control\*\* for the Stop button, integrated into the Dedupe tab layout.



\### Changed

\- \*\*Button layout and sizing\*\* updated for consistency across editions.

\- \*\*Minor UI polish\*\* to improve alignment and spacing.

\- \*\*Dedupe engine behaviour\*\* updated to support clean interruption.



\### Fixed

\- Small internal consistency issues in the dedupe workflow.
