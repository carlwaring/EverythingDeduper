# **Everything Deduper v1.2 — Improvements & Interruptible Auto‑Dedupe**

This release introduces the long‑awaited **Stop button** for Auto‑Dedupe, along with several workflow and UI refinements that make the dedupe process smoother, safer, and more predictable.

## 🚀 **What’s New**
- Added **Stop button** to interrupt Auto‑Dedupe safely  
- Added **interrupt logic** to the dedupe engine  
- Added **global running‑state flag** for auto‑mode  
- Added **UI control** for the Stop button in the Dedupe tab  

## 🎨 **Improvements**
- Updated button layout and sizing for consistency  
- Minor UI polish across the Dedupe tab  
- Improved dedupe engine behaviour to support clean interruption  

## 🛠️ **Fixes**
- Small internal workflow consistency fixes
