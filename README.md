# 📘 **Everything Deduper**  
*A fast, simple tool for finding and managing duplicate files.*

Everything Deduper is a pragmatic utility designed to solve a real‑world problem:  
**you have duplicate files scattered across drives, folders, backups, and archives — and you want them gone without stress, risk, or complexity.**

This tool focuses on clarity, safety, and workflow. No ads, no nags, no upsells, no nonsense.  
Just a clean interface, a predictable process, and a preview‑driven keeper selection system that puts you in control.

---

## ⭐ Why this tool exists
Most dedupers fall into one of two traps:

- **Too simple** — no previews, no control, no transparency  
- **Too complex** — too many toggles, too many modes, too much risk  

Everything Deduper sits in the middle:

- **Simple enough for non‑technical users**  
- **Powerful enough for technical users**  
- **Deterministic, interruptible, and safe**  
- **Preview‑first, not delete‑first**  
- **Designed for real workflows, not benchmarks**

It’s built to be *simple*, *predictable*, and *trustworthy*.

# 🧩 Features

## **Standard Edition Features**
These features are included in the Standard Edition:

### 🔍 **Audit Engine**
- Fast, threaded scanning  
- Hash‑based duplicate detection  
- Hash caching for faster re‑audits  
- Skips unreadable files with warnings  
- Generates both raw and readable audit logs  
- Push-button cleanup of old logs  

### 🗂️ **Preview‑Driven Keeper Selection**
- Auto‑selects a keeper using smart rules  
- Full preview window for each duplicate group  
- PDF preview (first page)  
- Image preview (scaled to panel)  
- Text preview (first 50 KB)  
- Office preview (DOCX/XLSX/PPTX text extraction)  
- Fallback hex preview for unknown formats  
- File metadata display (size, modified date)  
- File‑type filtering (All, PDF, JPG, PNG, DOCX, MP3)

### 🧠 **Smart Keeper Logic**
- Priority‑based keeper selection (folder names, paths)  
- Fallback to shallowest path  
- Deterministic behaviour  

### 🧹 **Dedupe Engine**
- Preview‑first workflow  
- Safe copy‑out model (never deletes originals) and keeps original directory stucture 
- Resume support via log parsing  
- Dry‑run mode  
- Clear logging of keepers and duplicates  

### 🖥️ **GUI**
- Clean, modern interface  
- Clear layout  
- Progress indicators  
- Interruptible operations  
- Settings persistence
